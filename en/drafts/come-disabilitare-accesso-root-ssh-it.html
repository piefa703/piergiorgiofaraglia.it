Come disabilitare accesso ssh come utente root


Published on 01 April 2015 in linux

Le librerie SSH installate sulle distribuzioni linux abilitano per default l'accesso dell'utente root sui server ssh. Sebbene questa impostazione sia di poco conto sulle intranet private o aziendali, essa assume una notevole importanza in termini di sicurezza su server pubblici. I seguenti passi servono per disabilitare tale accesso.

Per prima cosa, apri il file di configurazione del server ssh con permessi di root

sudo vi /etc/ssh/ssh_config

Modifica la direttiva PermitRootLogin impostandola a no

PermitRootLogin no

Salva il file e riavvia il server

sudo /etc/init.d/ssh restart
